8.0 - Images

Responsive Images approach using Picturefill. The picturefill.js script needs to be included somewhere in the page. Picturefill also requires some special markup to represent the image variations, differentiated by data-media attributes that act just like media queries in CSS.

You may also optionally put an image in conditional comments for browsers that don't support media queries, and a fallback in a noscript tag for browsers that donâ€™t have JavaScript enabled.

  • small - for mobile devices, postfix small images with '_small'
  • medium - for tablet devices, postfix medium images with '_med'
  • big - for desktop devices, postfix big images with '_big'

8.1 - Retina Images

Picturefill also supports HD(Retina) image replacement.

  • small HD - for mobile devices, postfix small retina images with '_small@2x'
  • medium HD - for tablet devices, postfix medium retina images with '_med@2x'
  • big HD - for desktop devices, postfix big retina images with '_big@2x'

Images only show up in HD devices
